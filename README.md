# 🖼️ area_hunt : 일상 속 숨겨진 장소를 찾아서

## http://area-hunt.s3-website.ap-northeast-2.amazonaws.com/

## 다른 사람들은 모르는, 주변의 멋진 장소들을 공유하는 서비스<br>

- 기간: 23.06.04 ~ 23.06.18 / 23.07 ~ 23.08

## 서비스 기능 요약

|       기능       | 설명                                                                       | 경로       |
| :--------------: | :------------------------------------------------------------------------- | :--------- |
|       메인       | 이 서비스를 간략하게 소개하는 페이지입니다.                                | /          |
|     회원가입     | 아이디, 비밀번호, 이메일, 전화번호를 입력하여 일반 회원 가입이 가능합니다. | /join      |
|      로그인      | 아이디, 비밀번호를 통해 로그인할 수 있습니다.                              | /login     |
|       등록       | 장소의 사진, 이름, 주소, 위치, 간단한 소개글을 등록할 수 있습니다.         | /regist    |
|      리스트      | 등록된 장소을 확인할 수 있습니다.                                          | /list      |
|      내정보      | 자신의 정보를 확인할 수 있습니다.                                          | /mypage    |
|       검색       | 검색한 단어에 해당하는 장소들을 확인할 수 있습니다.                        | /search    |
|      지역별      | 등록된 장소들을 지역별로 확인할 수 있습니다.                               | /locations |
|     notfound     | 잘못된 접근임을 알려줍니다.                                                | /notfound  |
| 지역 상세 페이지 | 등록된 장소의 상세 정보 및 후기를 보여줍니다.                              | /:id       |
